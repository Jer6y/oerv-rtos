# oerv-rtos小组协作仓库
## 仓库说明

本仓库为 **oerv-rtos** 小队协作仓库， 工作于 openEuler 仓库中的 UniProton 仓库 ， MCS 仓库 以及 embedded-ci 仓库等。工作内容为给openEuler-embedded 的原生项目提供 RISCV 支持。目前基础核心项目为 openEuler 的原生RTOS UniProton。
